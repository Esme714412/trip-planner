# trip-planner

① 手機縮放 — index.html 加 user-scalable=no
② 旅行日期區間 — 建立行程時可選日期或跳過；日期顯示在 header 下方可點擊編輯；新增景點時若已設定日期區間，日期欄位改為「第N天（M/D 週）」下拉選單
③ 暱稱同步 — 新增分享成員時，優先用對方的 nickname，再 displayName，再 email 前綴
④ 參與人員保護 — 第一位成員顯示 👑 無法被刪除；只有行程擁有者（isOwner）才能看到刪除按鈕
⑤ 回到頂部按鈕 — 滑動超過 300px 後右下角浮出紫色按鈕
⑥ 離線提示 — 斷線時頂部顯示橙色橫幅
⑦ 刪除行程保護 — 需輸入完整行程名稱才能確認刪除
⑧ 唯讀模式 — viewer 進入時：隱藏分享、設定、編輯切換、新增景點、記一筆、清單新增刪除等所有編輯按鈕；頂部顯示「👁 瀏覽」標籤
