# 旅行計畫助手 Trip Planner

一個基於 React + Firebase 的旅遊行程管理 Web App，支援多人共享、即時同步、記帳分帳。

---

## 技術架構

- **前端**：React + Vite + Tailwind CSS
- **後端**：Firebase Authentication（Google 登入）+ Firestore（即時資料庫）
- **部署**：Netlify（GitHub 自動部署）

---

## Firestore 資料結構

```
users/{uid}/trips/{tripId}        # 自己的行程
sharedTrips/{tripId}              # 共享行程索引（editors[], viewers[]）
userProfiles/{uid}                # 使用者資料（email, displayName, nickname）
allowedUsers/{email}              # 白名單
```

## Markdown 匯入格式

```markdown
# 旅程名稱（選填）

## 2025-05-01
- time: 09:00
  title: 關西國際機場          # 必填
  location: 關西國際機場
  transport: 關空特急 Haruka
  transport_departure: 09:30
  transport_duration: 75分鐘
  transport_price: 3600
  transport_url: https://www.westjr.co.jp/
  transport_ticket: true       # true = 需提前購票，自動加入行前清單
  hours: 09:00-22:00
  website: https://...
  notes: 注意事項備註
  tickets: 門票資訊
  todo:
    - 行前待辦事項
  shopping:
    - 購物項目

## 行前清單
- [ ] 未完成事項
- [x] 已完成事項
```

**注意**：
- 只有 `title` 為必填，其餘欄位皆可省略
- 日期格式：`YYYY-MM-DD`
- 匯入的內容會附加到現有行程後面，不會覆蓋
